IoT Safety Project

🔥 Fire Detection
System — ESP8266

Rapid-response flame & smoke detection using DHT11 + MQ-2 sensors with real-time audible/visual alerts and optional WiFi connectivity.

01

Aim & GitHub

To design and develop an IoT-based Fire Detection System using the ESP8266 microcontroller integrated with a DHT11 temperature sensor and an MQ-2 smoke/gas sensor — ensuring rapid detection of fire hazards through real-time monitoring, audible buzzer and visual LED alarms, and optional WiFi-based notifications to enhance safety in homes, labs, and industrial environments.

github.com/edwiee/FIRE-DETECTION-SYSTEM---ESP8266
02

Problem Statement

⚠️

The Challenge

Fire accidents cause devastating loss of life and property. Traditional alarm systems rely on single-parameter detection, suffer from delayed response, and lack IoT integration for remote alerts.

💡

The Need

A low-cost, multi-parameter fire detection solution that simultaneously monitors temperature AND smoke levels, triggers immediate alarms, and can be deployed anywhere affordably.

📊

Current Gaps

Existing solutions are either too expensive, require professional installation, don't provide real-time data logging, or cannot integrate with smart home/IoT platforms.

🎯

Proposed Solution

ESP8266-based system with DHT11 + MQ-2 dual sensing, buzzer/LED alerts, LCD display, configurable thresholds, WiFi connectivity, and open-source codebase.

↓

↓

↓
YES — HAZARD
🔴 RED LED ON
🔔 Buzzer ON
LCD: FIRE ALERT!!
NO — NORMAL
🟢 GREEN LED ON
LCD: Temp + Smoke
Serial: OK
↓
🔘 Button Pressed? → Silence Buzzer
↓
↩ Return to Loop

// ── Threshold Settings ──────────────────────────────
#define TEMP_THRESHOLD    50.0   // °C — fire warning level
#define SMOKE_THRESHOLD   400    // ADC value (0-1023)
#define SAMPLE_INTERVAL   1000   // ms between readings

// ── Read & Evaluate ──────────────────────────────────
void readAndEvaluate() {
  float humidity    = dht.readHumidity();
  float temperature = dht.readTemperature();
  int   smokeLevel  = analogRead(SMOKE_PIN);

  bool tempAlert  = (temperature >= TEMP_THRESHOLD);
  bool smokeAlert = (smokeLevel  >= SMOKE_THRESHOLD);

  if (tempAlert || smokeAlert) {
    triggerAlert(temperature, smokeLevel);
  } else {
    clearAlert(temperature, smokeLevel);
  }
  updateLCD(temperature, humidity, smokeLevel, tempAlert||smokeAlert);
}

// ── Trigger Alert ────────────────────────────────────
void triggerAlert(float temp, int smoke) {
  alertActive = true;
  digitalWrite(LED_RED_PIN,   HIGH);
  digitalWrite(LED_GREEN_PIN, LOW);
  tone(BUZZER_PIN, BUZZER_FREQ, 200);
  Serial.println(F("[ALERT] *** FIRE DETECTED ***"));
}
